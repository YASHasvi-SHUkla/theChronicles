# TheChronicles

Welcome to TheChronicles - your ultimate source for up-to-date news from around the world! This application is built using React and utilizes the News API along with React Hooks to deliver a seamless and engaging news browsing experience.

## Features

- **Real-time Updates**: Access the latest news articles from various sources instantly.
- **Customizable Categories**: Explore news articles based on your interests and preferences.
- **Responsive Design**: Enjoy a consistent and intuitive user interface across different devices.
- **Interactive Interface**: Engage with news articles through interactive features and smooth navigation.
- **Search Functionality**: Easily search for specific topics or articles to find relevant information quickly.

## Installation

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Obtain an API key from [News API](https://newsapi.org/) and replace `YOUR_API_KEY` in the `.env` file with your actual API key.
5. Run `npm start` to start the development server.
6. Open your browser and visit `http://localhost:3000` to access NewsHub.

## Technologies Used

- React: JavaScript library for building user interfaces.
- News API: Source for real-time news articles from various publishers.
- React Hooks: Simplify state management and side effects in functional components.
